This is the highest-priority blocker — without this, no other test tasks can proceed.

## Steps

1. **Install dependencies** in `packages/intelli-mock-core/`:
   ```
   pnpm add -D vitest supertest @types/supertest @faker-js/faker
   ```
   Also add `vitest` as a devDependency at the root if not present.

2. **Create `vitest.config.ts`** in `packages/intelli-mock-core/`:
   - Node environment, globals enabled
   - setupFiles pointing to `./test/setup.ts`
   - include pattern: `test/**/*.test.ts`
   - v8 coverage provider, text + lcov reporters
   - Thresholds: 80% lines, 75% branches, 80% functions, 80% statements
   - passWithNoTests: true (so CI doesn't fail before tests exist)

3. **Create `tsconfig.test.json`** in `packages/intelli-mock-core/test/`:
   - Extend parent tsconfig
   - Add `vitest/globals` to types
   - Include `../src/**/*.ts` and `./**/*.ts`

4. **Create `test/setup.ts`**:
   - Import `vi`, `beforeEach`, `afterEach` from `vitest`
   - In `beforeEach`: call `resetConfig()` from env.ts, set safe default env vars
   - In `afterEach`: call `resetConfig()` again, clear any stubbed env vars

5. **Create `test/helpers/fixtures.ts`**:
   - Factory functions for `Tenant`, `MockEndpoint`, `User` entities
   - Use `@faker-js/faker` with seed `42` for deterministic output
   - Each factory accepts `Partial<Entity>` overrides

6. **Create `test/helpers/mock-container.ts`**:
   - `resetContainer()` function that clears tsyringe's internal `_registry` and `_instances`

7. **Create `vitest.workspace.ts`** at repo root:
   - Define workspace pointing to `packages/intelli-mock-core/vitest.config.ts`

8. **Add test scripts**:
   - Root `package.json`: `test: "pnpm -r test"`, `test:watch`, `test:coverage`
   - Core `package.json`: `test: "vitest run"`, `test:watch`, `test:coverage`

9. **Verify**: Run `pnpm test` — should output "0 tests" with exit code 0.
