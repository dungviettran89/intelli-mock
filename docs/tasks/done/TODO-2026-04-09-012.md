# Task: AI Service + Prompts (Local Ollama Integration)

**Status:** completed
**Created:** 2026-04-09
**Priority:** high
**Estimated Time:** 20-30 mins

## Context
Phase 3: AI Engine is progressing well. Script Service + Validator (TODO-011) is complete. Now we need the **AI Service** that integrates with Vercel AI SDK to communicate with the local Ollama OpenAI-compatible endpoint (`http://localhost:11434/v1`) using the `gemma4:31b-cloud` model. This service will generate JavaScript mock scripts from sample request/response pairs.

## Scope

- [x] `AIService` — Vercel AI SDK integration with OpenAI-compatible provider
- [x] `prompts.ts` — System and user prompt templates for script generation
- [x] `generateScript(samples, model?)` — Main method to generate scripts from samples
- [x] Configuration for local Ollama endpoint (`AI_BASE_URL`, `AI_MODEL`)
- [x] Unit tests for AI Service (fully mocked, no real Ollama calls)

### Out of Scope

## Steps
1. **Install `ai` package** (Vercel AI SDK) — already listed in PROGRESS.md as needed for Phase 3

2. **Create `prompts.ts`** (`src/modules/ai/prompts.ts`)
   - System prompt: "You are an expert JavaScript developer. Generate Express-style mock handlers..."
   - User prompt template: Compose from sample pairs (request → response format)
   - Include context about available utils (delay, random, pick, oneOf)

3. **Create `AIService`** (`src/modules/ai/ai.service.ts`)
   - Use `createOpenAI()` provider with custom `baseURL` from env
   - Call `generateText()` with composed prompt
   - Return generated code with metadata (model, tokens used)
   - Handle errors gracefully (rate limits, model unavailable)

4. **Add AI config to env.ts** (if not present)
   - `AI_BASE_URL` (default: `http://localhost:11434/v1`)
   - `AI_API_KEY` (default: `ollama`)
   - `AI_MODEL` (default: `gemma4:31b-cloud`)

5. **Register in DI container** (`container.ts`)

6. **Write unit tests** (`test/modules/ai/ai.service.test.ts` + `test/modules/ai/prompts.test.ts`)
   - Mock Vercel AI SDK completely (vi.mock('ai'))
   - Test successful generation, error handling, prompt composition
   - Target: 12+ tests total
